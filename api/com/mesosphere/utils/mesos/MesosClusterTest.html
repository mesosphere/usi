MesosClusterTest 

trait MesosClusterTest extends Suite with ZookeeperServerTest with MesosTest with ScalaFutures with Eventually with Matchers

Basic trait to include in tests. It comes with Zookeeper (in-memory) and a minimal Mesos cluster: one master, one agent with default configuration parameters. To use, simply extend your test class from it.
